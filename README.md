# NTT DATA — APAC Data Centre Leadership Summit 2026

Frontend for the registration/waitlist landing page. Static HTML/CSS/JS, no build step or dependencies.

## Structure

- `index.html` — page markup (hero, why-attend, speakers, agenda, registration, footer)
- `css/styles.css` — design system and responsive layout
- `js/main.js` — mobile nav, registration drawer, form validation, scroll reveals
- `assets/` — images
- `ntt-waitlist-page.pdf` / `.png` — original design reference

## Running locally

Open `index.html` directly in a browser — no server or build required.
